# "创作者工具箱" (Creator Tools) V2.0 产品需求文档 (PRD)

**版本:** 2.0
**更新日期:** 2025年1月15日

## 1. 项目愿景与目标

### 1.1. 愿景
打造一个专为自媒体人、内容创作者设计的高效、免费、尊重隐私的在线工具集合，成为他们创作流程中不可或缺的“瑞士军刀”。

### 1.2. 核心原则
- **无需登录:** 所有核心功能即用即走，无任何使用门槛。
- **极致速度:** 网站和工具响应迅速，不浪费用户时间。
- **尊重隐私:** 尽可能在浏览器本地完成所有计算，不上传用户数据。
- **简洁设计:** 界面清晰，无广告，无干扰，专注于工具本身。

## 2. 目标用户画像
- **小红 (图文博主):** 经常需要处理图片（压缩、格式转换）、为文章想标题。
- **阿伟 (视频UP主):** 经常需要下载参考素材（封面图）、为视频想标题。
- **李静 (新媒体运营):** 日常需要处理各种图片、统计文案字数、辅助内容创作。

## 3. V1.2 功能范围
V1.2 版本在原有5个核心工具基础上，新增配音工具和内容提取工具，进一步完善创作者工具生态。

1.  **图片压缩器 (Image Compressor)**
2.  **爆款标题生成器 (Headline Generator)**
3.  **YouTube/B站封面下载器 (Thumbnail Downloader)**
4.  **字数统计与分析器 (Word Counter & Analyzer)**
5.  **图片格式转换器 (Image Format Converter)**
6.  **配音工具 (Voice Generator)**
7.  **内容提取工具 (Content Extractor)**

## 4. 功能详细需求

### 4.1. 图片压缩器
- **用户故事:** 作为创作者，我希望能批量上传图片，在不严重损失画质的情况下，显著减小文件大小，以便提升网页加载速度。
- **功能点:**
    - [P0] 支持拖拽和点击上传多张图片 (JPG, PNG, WebP)。
    - [P0] 上传后自动开始压缩。
    - [P0] 以卡片形式展示每张图片的结果，包含：原大小、压缩后大小、压缩率。
    - [P0] 提供直观的图片质量对比功能（如鼠标悬浮切换）。
    - [P0] 提供单个文件下载和“一键打包下载所有(.zip)”功能。
    - [P0] 提供“清空全部”功能。
    - [P1] 提供压缩等级调整滑块。

### 4.2. 爆款标题生成器
- **用户故事:** 作为创作者，我希望能输入主题关键词并选择平台，AI能为我生成一批吸引人的标题，以解决我的“标题困难症”。
- **功能点:**
    - [P0] 提供文本框输入核心关键词。
    - [P0] 提供目标平台选择（如公众号、小红书、B站等）。
    - [P0] 提供“一键生成”按钮，并显示加载状态。
    - [P0] 结果按平台分组展示。
    - [P0] 每个标题提供“一键复制”按钮。
    - [P1] 提供标题风格选择（如专业、夸张、疑问式）。
    - [P1] 提供“收藏”和“换一批”功能。

### 4.3. YouTube/B站封面下载器
- **用户故事:** 作为内容研究者，我希望能粘贴一个视频链接，然后一键下载该视频的高清封面图，用于我的素材收集和分析。
- **功能点:**
    - [P0] 提供输入框用于粘贴YouTube或B站的视频链接。
    - [P0] 粘贴后自动解析，或提供“解析”按钮。
    - [P0] 展示解析出的不同分辨率的封面图（如HD, SD等）。
    - [P0] 每张图片提供“下载”按钮。
    - [P1] 对无效链接或无法解析的视频给出清晰错误提示。

### 4.4. 字数统计与分析器
- **用户故事:** 作为写手，我希望能粘贴一段文本，系统能立即告诉我它的总字数、字符数、段落数和预计阅读时间，以满足稿件要求。
- **功能点:**
    - [P0] 提供一个大的文本输入区域。
    - [P0] 在用户输入时，实时更新统计数据（字数、字符数、段落数、预计阅读时间）。
    - [P0] 提供“一键清空”按钮。

### 4.5. 图片格式转换器
- **用户故事:** 作为创作者，我希望能方便地将图片在不同格式间转换，以满足特定平台的上传要求。
- **功能点:**
    - [P0] 支持拖拽和点击上传一张或多张图片。
    - [P0] 支持的输入格式：JPG, PNG, WebP。
    - [P0] 提供目标格式选择：JPG, PNG, WebP。
    - [P0] 转换完成后，提供单个文件下载和“一键打包下载所有(.zip)”功能。
    - [P1] 如果转换目标是JPG/WebP，提供图片质量设置滑块。

### 4.6. 配音工具
- **用户故事:** 作为内容创作者，我希望能将文本转换为语音，支持多种音色和语言，特别是能够克隆特定音色，以满足不同场景的配音需求。
- **功能点:**
    - [P0] 提供文本输入区域，支持长文本输入。
    - [P0] 支持两种语音模式：系统语音和AI音色克隆。
    - [P0] 系统语音模式：提供多种内置语音选择，支持语速、音调、音量调节。
    - [P0] AI音色克隆模式：支持上传10-60秒音频样本进行音色克隆。
    - [P0] 支持17+种语言，包括中文、英文、日文、韩文等。
    - [P0] 生成高质量MP3格式音频文件。
    - [P0] 提供音频播放预览和下载功能。
    - [P1] 克隆音色本地存储，支持重复使用。
    - [P1] 提供音色管理功能（重命名、删除）。

### 4.7. 内容提取工具
- **用户故事:** 作为内容研究者，我希望能够从主流自媒体平台提取视频、音频、图文内容，并支持下载，用于内容分析和素材收集。
- **功能点:**
    - [P0] 支持多平台URL解析：YouTube、Bilibili、抖音、小红书、微信公众号、X(Twitter)、小宇宙、喜马拉雅。
    - [P0] 智能平台检测：自动识别输入URL所属平台。
    - [P0] 内容信息展示：标题、作者、发布时间、播放量、点赞数等元数据。
    - [P0] 多格式下载支持：视频(1080p/720p/480p)、音频、缩略图。
    - [P0] 下载进度显示和状态反馈。
    - [P0] 支持批量URL处理。
    - [P1] 内容预览功能（缩略图、简介）。
    - [P1] 下载历史记录。

## 5. 非功能性需求
- **性能:** 所有工具页面首次有效绘制 (FCP) 应小于 1.5s。
- **SEO:** 每个工具页面都应有独立的、对搜索引擎友好的`<title>`和`<meta description>`。
- **可访问性 (A11y):** 所有交互元素都应能通过键盘访问并有适当的语义化标签。
